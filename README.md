Мои проектные работы для курса Яндекс Практикума ["Обработка естественного языка — NLP"](https://practicum.yandex.ru/natural-language-processing/).

1. [`multitask_model`](multitask_model.ipynb) — разработка многозадачной модели для классификации токенов (NER) и распознавания логических отношений в тексте.
2. [`llm_pretrain_sft`](llm_pretrain_sft.ipynb) — pretrain LLM на корпусе текстов русской литературы и SFT модели для ответов на вопросы.
3. [`retrieval_system`](retrieval_system.ipynb) — разработка системы поиска научных статей по тексту аннотации с использованием векторного хранилища.
4. [`clip_search_model`](clip_search_model.ipynb) — разработка системы поиска изображений товаров по текстовому описанию на основе дообученной модели CLIP.
